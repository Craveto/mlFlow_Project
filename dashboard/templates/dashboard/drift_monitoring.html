Interpretation
How this page reads drift

This page uses lightweight heuristics so you can quickly spot instability before moving to a full monitoring stack.

Volatility ratio is the main trigger If recent 24-hour volatility rises well above the 7-day baseline, the page raises the drift status.
Price displacement gives extra context A large move away from the 21-hour trend can indicate a regime shift even if the latest hourly return is small.
Use this as a front-door monitor It is intentionally simple and readable, so you can decide when deeper drift analysis is needed.
